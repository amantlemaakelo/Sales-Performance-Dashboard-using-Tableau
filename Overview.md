# Sales Performance Dashboard using Tableau

## Project Overview

This project presents an interactive Tableau dashboard developed to analyse a US retail sales dataset obtained from Kaggle. The dashboard provides insights into sales performance across regions, product categories, products, profitability, and time through interactive visualisations and data storytelling.

## Objectives

* Analyse retail sales data using Tableau.
* Develop calculated fields and KPIs.
* Build an interactive dashboard with dashboard actions and filters.
* Identify key business insights from the data.
* Communicate findings through data storytelling.

## Dashboard Features

* KPI Summary (Revenue, Profit, Orders and Profit Margin)
* Revenue by Region
* Revenue by Category
* Revenue by Subcategory
* Top Products by Revenue
* Monthly Revenue Trend
* Profit vs Revenue Analysis
* Interactive Dashboard Actions

## Tools Used

* Tableau Desktop
* Microsoft Excel / CSV Dataset
* Kaggle Dataset

## Dataset

The dataset used in this project is the Product Sales Dataset (2023–2024) obtained from Kaggle and contains approximately 200,000 retail sales records.

## Key Insights

* Electronics generated the highest revenue.
* The East region recorded the strongest sales performance.
* Revenue peaked during the fourth quarter of the year.
* A small number of products contributed a large share of total revenue.
* Higher revenue did not always correspond to higher profitability.
